## Overview

This document defines Alibaba-specific extensions to the [GenAI Skill semantic conventions](../gen-ai-spans.md).
GenAI skill represents a reusable, self-contained capability that can be invoked by agents or workflows.
Skills encapsulate specific functionalities such as code review, data analysis, or task execution.
These extensions add Alibaba infrastructure context to skill telemetry.

### Span name

**Span name** SHOULD be `invoke_skill {gen_ai.skill.name}` if `gen_ai.skill.name` is readily available.
When `gen_ai.skill.name` is not available, it SHOULD be `invoke_skill`.

### Alibaba Skill Span Attributes

In addition to the standard [GenAI Skill attributes](../gen-ai-spans.md), the following
Alibaba-specific attributes SHOULD be populated for skill spans running on Alibaba infrastructure:

| Attribute | Type | Description | Requirement Level |
| --- | --- | --- | --- |
| `alibaba.base.env` | string | Environment identifier | Recommended |
| `alibaba.base.idc` | string | Data center identifier | Recommended |
| `alibaba.user.data` | string | User-defined custom data attached to the skill invocation | Opt-in |
| `alibaba.experiment.id` | string | RL experiment identifier (for RL-trained skills) | Conditionally Required: if applicable |
| `alibaba.group.id` | string | RL group identifier (for RL-trained skills) | Conditionally Required: if applicable |
| `alibaba.job.id` | string | RL job identifier (for RL-trained skills) | Conditionally Required: if applicable |
| `alibaba.instance.id` | string | RL instance identifier (for RL-trained skills) | Conditionally Required: if applicable |
