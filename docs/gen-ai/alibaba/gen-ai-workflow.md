## Overview

This document defines Alibaba-specific extensions to the [GenAI Workflow semantic conventions](../gen-ai-spans.md).
These extensions add Alibaba infrastructure context to workflow telemetry.

### Span name

**Span name** SHOULD be `invoke_workflow {gen_ai.workflow.name}` if `gen_ai.workflow.name` is readily available.
When `gen_ai.workflow.name` is not available, it SHOULD be `invoke_workflow`.

### Span kind

**Span kind** SHOULD be `CLIENT` and MAY be set to `INTERNAL` on spans representing invocation of workflows running in the same process.

### Alibaba Workflow Span Attributes

In addition to the standard [GenAI Workflow attributes](../gen-ai-spans.md), the following
Alibaba-specific attributes SHOULD be populated for workflow spans running on Alibaba infrastructure:

| Attribute | Type | Description | Requirement Level |
| --- | --- | --- | --- |
| `alibaba.base.env` | string | Environment identifier | Recommended |
| `alibaba.base.idc` | string | Data center identifier | Recommended |
| `alibaba.user.data` | string | User-defined custom data attached to the workflow | Opt-in |
| `alibaba.experiment.id` | string | RL experiment identifier (for RL-based workflows) | Conditionally Required: if applicable |
| `alibaba.group.id` | string | RL group identifier (for RL-based workflows) | Conditionally Required: if applicable |
| `alibaba.job.id` | string | RL job identifier (for RL-based workflows) | Conditionally Required: if applicable |

## Workflow Metric

### Metric: `gen_ai.workflow.duration`

This metric represents the duration of GenAI workflow invocations.

| Name | Instrument Type | Unit |
| --- | --- | --- |
| `gen_ai.workflow.duration` | Histogram | ms |

| Attribute | Requirement Level |
| --- | --- |
| `gen_ai.workflow.name` | Required |
| `error.type` | Conditionally Required: if error |
| `alibaba.base.env` | Recommended |
| `alibaba.base.idc` | Recommended |
